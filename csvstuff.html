Importing Data

You always want to load your packages at the top of your script/ document.

library(tidyverse)
library(readr)

You can have functions from R packages (like dplyr), but you can also write R functions yourself. Load the custom function clean_keyence_csv…

# Load clean_keyence_csv()
source("custom_functions/clean_keyence_csv.R")

Use the function like this…

  1. Get your file names
file_paths <- list.files(path = "data/raw", pattern = "\\.csv$", full.names = TRUE)
  1. Process it all into one tibble (similar to data.frame)
  compiled_data <- suppressWarnings(
    file_paths %>%
      set_names(basename(.)) %>%
      map(clean_keyence_csv) %>%
      list_rbind(names_to = "source_file")
  )

Then, you can check out your result like this…

View(compiled_data)

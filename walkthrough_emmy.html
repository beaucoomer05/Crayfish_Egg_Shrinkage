Analysis

Experimental Design

10 Procambarus clarkii individuals, each with eggs split across 4 preservation treatments (EtOH80, EtOH95, Iso70, Formalin10). There are not necessarily the same number of eggs placed in each preservation treatment from a given individual.

You can check to make sure the data we’re using matches what it should be…

[1] 10
[1] "EtOH80"     "EtOH95"     "Formalin10" "Iso70"

The variables of this study are…

Type Variable(s)
Independent Treatment, Individual*
Dependent Egg size (diameter in mm)
Controlled Species, Individual*

The assumptions for most parametric tests are:

  1. Normality
  2. Equal variance
  3. Independence - data is sampled randomly and independently (not biased)
  4. No outliers (no extremes)

Our experimental design violates the assumption of independence — there are multiple egg measurements per individual. We have repeated measures, which, if ignored, undermines the statistical sanctity of the results (i.e., your interpretation would not be valid if you just ignore it).

To account for repeated measures, you can explicitly model the correlation between that of the dependent variable (egg size (continuous)) and the source of the repeated measure (individual) using a linear mixed-effect model or repeated-measures ANOVA.

Since the eggs from each individual were spread across the four treatments, the experimental design is crossed. There were not an equal number of eggs per individual/ per treatment, it is also unbalanced.

Linear Mixed-Effect Model (LME)

The R package {lme4} is widely used for linear mixed effect modelling. The base R function lm() is for linear models, but it can’t handle experimental designs with random factors, so that’s why we need {lme4}.

There are a couple considerations in the lmer() function that we’ll want to overload. The R package {lmerTest} does this, so we’ll load it too.

library(lme4)
library(lmerTest)
TipLearn more about these models!

You can learn more about linear models in Chapter 18 of The Analysis of Biological Data by Whitlock and Schluter (3rd Edition, ISBN-10: 131922623X) — (R Code for Examples).

Dynamic Ecology has a interesting blog post on effects… Is it a fixed or random effect?

There’s also a great chapter on Linear Mixed Models from a Washington U course.

For the model, the user defines the response, fixed effect, and random effect(s).

\[\mathrm{Egg Diameter}_{ij} = \mathrm{Treatment}_i + (1| \mathrm{Individual}_j)\]

Since every individual has eggs across all four treatments, the model can cleanly separate the effect of the treatment from the unique characters of the individual. lmer uses maximum likelihood (ML) estimates that naturally weight individuals with more eggs heavier for calculating individual traits while still accurately calculating the overall effect of treatment. The model also pulls/ “shrinks” extreme averages toward the global mean, which prevents a single individual with few eggs from potentially skewing the result.

df_analysis$ind_num <- as.factor(df_analysis$ind_num)
df_analysis$treatment <- as.factor(df_analysis$treatment)
# Create the model
mod_full <- lmer(egg_diameter_mm ~ treatment + (1|ind_num) + (1|treatment:ind_num), data = df_analysis)

Going Further, the Conservative Route

Having all the treatments be statistically significant from one another is just down right weird. We know that the residuals of the model were skewed, even after the egg diameters were log transformed. Since the normality assumption is violated, the p-values are probably artificially deflated. We need to know if our treatment differences are real, so we can use a generalized linear mixed model (GLMM) with a link function.

robust_mod <- glmer(egg_diameter_mm ~ treatment + (1|ind_num), family = Gamma(link = "log"), 
                    data = df_analysis)
library(car)
library(DHARMa)
Anova(robust_mod, type = "III")

(Intercept) 1042.5  1  < 2.2e-16 ***

---

residuals_sim <- simulateResiduals(fittedModel = robust_mod)
plot(residuals_sim)
Note

The Gamma distribution called from family = tells the function to expect the skewed data. It adjusts the standard errors, so the pairwise comparisons might be more realistic.

Robust Model: nmle

# 1. Load the packages
library(nlme)
library(car)      # For Type III Anova
library(emmeans)  # For post-hoc pairwise comparisons
# 2. Fit the model using Generalized Least Squares via lme
# random = ~ 1 | ind_num/treatment builds the nested random interaction term
# weights = varIdent(form = ~ 1 | treatment) models unique variances per group
nlme_model <- lme(fixed = egg_diameter_mm ~ treatment, 
                  random = ~ 1 | ind_num/treatment, 
                  weights = varIdent(form = ~ 1 | treatment), 
                  data = df_sub,
                  method = "REML")

Visualization

The best (statistically sound model) is the robust model from {nmle}.

  • Accomodates randomized block design with subsampling and unequal design
  • Potential outliers were removed using a Cook’s distance threshold
  • Used varIdent function to address heteroscedasticity
  • Individuals and treatment-by-individual interaction were modeled as nested random effects.
  • Significant effect of treatment on egg diameter (Type III ANOVA)
  • Post-hoc pairwise comparisons with Tukey adjustments
    • Iso70 had larger egg diameters than all other groups
    • No difference between EtOH80 and EtOH95
    • Formalin10 resulted in significantly smaller egg sizes than all other treatments.

# Analysis

## Experimental Design

**10** *Procambarus clarkii* individuals, each with eggs split across **4** preservation treatments (EtOH80, EtOH95, Iso70, Formalin10). There are not necessarily the same number of eggs placed in each preservation treatment from a given individual.

You can check to make sure the data we're using matches what it should be...

```{r}
#| echo: false

# Number of individuals
length(unique(df_analysis$ind_num))

# List of treatments
unique(df_analysis$treatment)
```

The variables of this study are...

| Type | Variable(s)  |
| ---- | ------------ |
| Independent | Treatment, Individual* | 
| Dependent | Egg size (diameter in mm) | 
| Controlled | Species, Individual* | 

The [assumptions](https://www.statology.org/parametric-tests-assumptions/) for most parametric tests are:

1. Normality
2. Equal variance
3. Independence - data is sampled randomly and independently (not biased)
4. No outliers (no extremes)

Our experimental design violates the assumption of independence --- there are multiple egg measurements per individual. We have **repeated measures**, which, if ignored, undermines the statistical sanctity of the results (i.e., your interpretation would not be valid if you just ignore it). 

To account for repeated measures, you can explicitly model the correlation between that of the dependent variable (egg size (continuous)) and the source of the repeated measure (individual) using a **linear mixed-effect model** or **repeated-measures ANOVA**. 

Since the eggs from each individual were spread across the four treatments, the experimental design is **crossed**. There were not an equal number of eggs per individual/ per treatment, it is also **unbalanced**.

## Linear Mixed-Effect Model (LME)

The R package `{lme4}` is widely used for linear mixed effect modelling. The base R function `lm()` is for linear models, but it can't handle experimental designs with random factors, so that's why we need `{lme4}`.

There are a couple considerations in the `lmer()` function that we'll want to overload. The R package `{lmerTest}` does this, so we'll load it too.

```{r}
#| message: false
#| warning: false
library(lme4)
library(lmerTest)
```

::: {.callout-tip collapse="true" title="Learn more about these models!"}
You can learn more about linear models in Chapter 18 of *The Analysis of Biological Data* by Whitlock and Schluter (3rd Edition, ISBN-10: 131922623X) --- [(R Code for Examples)](https://whitlockschluter3e.zoology.ubc.ca/RExamples/Rcode_Chapter_18.html). 

Dynamic Ecology has a interesting blog post on effects... [Is it a fixed or random effect?](https://dynamicecology.wordpress.com/2015/11/04/is-it-a-fixed-or-random-effect/)

There's also a great chapter on [**Linear Mixed Models** from a Washington U course](https://courses.washington.edu/psy524a/_book/linear-mixed-models.html).
:::

For the model, the user defines the response, fixed effect, and random effect(s). 

$$\mathrm{Egg Diameter}_{ij} = \mathrm{Treatment}_i + (1| \mathrm{Individual}_j)$$

Since every individual has eggs across all four treatments, the model can cleanly separate the effect of the treatment from the unique characters of the individual. `lmer` uses **maximum likelihood** (ML) estimates that naturally weight individuals with more eggs heavier for calculating individual traits while still accurately calculating the overall effect of treatment. The model also pulls/ "shrinks" extreme averages toward the global mean, which prevents a single individual with few eggs from potentially skewing the result.

```{r}
df_analysis$ind_num <- as.factor(df_analysis$ind_num)
df_analysis$treatment <- as.factor(df_analysis$treatment)
```

```{r}
# Create the model
mod_full <- lmer(egg_diameter_mm ~ treatment + (1|ind_num) + (1|treatment:ind_num), data = df_analysis)

# View the result
mod_full
```

### Overall Treatment Effect

Passing the output of `lmer()` to `anova()` gives you the ANOVA table.

```{r}
anova(mod_full)
```

P < 0.05, so **treatment significantly affects the egg diameter**, but it does not give more information as to *which* treatments differ. We'll need to do pairwise comparisons to figure that out.

### Pairwise Comparisons

The treatment is significant, so we can use the R package `{emmeans}` (estimated marginal means). 

```{r}

#| warning: false
library(emmeans)
```

```{r}
#| message: false
emm <- emmeans(mod_full, ~ treatment)
```

```{r}
pairs(emm, adjust = "tukey")
```

::: {.callout-note}
Look at `p.value` in the output. Any value under 0.05 means the two treatments (in `contrast`) are statistically different.
:::

Mean egg diameter differed in all preservative types.

```{r}
summary(emm)
```

Linear mixed models assume the residuals are normally distributed and have equal variance. The study design is unbalanced, so verifying assumptions is even more important.

```{r}
# Check variance
plot(mod_full)

# Visual check for normality of residuals
qqnorm(residuals(mod_full), main = "Q-Q Plot of Residuals")
qqline(residuals(mod_full), col = "red", lwd = 2)

# Statistical check for normality of residuals
shapiro.test(residuals(mod_full))
```

The residuals are falling pretty darn far from the line, especially on the lower end. Since we are dealing with linear measurement data, this kind of normality violation is kind of expected and we can try to transform the data and re-run the model...

```{r}
df_analysis$log_egg_diameter_mm <- log(df_analysis$egg_diameter_mm) + 1
```

```{r}
mod_log <- lmer(log_egg_diameter_mm ~ treatment + (1|ind_num) + (1|treatment:ind_num), data = df_analysis)
```

Now we can check the assumptions again before moving forward...

```{r}
# Check variance
plot(mod_log)

# Check normality of residuals
qqnorm(residuals(mod_log), main = "Q-Q Plot of Residuals (log-transformation)")
qqline(residuals(mod_log), col = "red", lwd = 2)
```

Still, normality is not fantastic, but it's slightly better.

Another assumption is equal variances. We can check this visually, or with a statistical test.

```{r}
#| include: false
#| eval: true
library(car)
```

```{r}
# Visual Check: Points should be random cloud/ smear
plot(mod_log, type = c("p", "smooth"),
     main = "Residuals vs Fitted Values",
     xlab = "Fitted Values", ylab = "Residuals")

# Levene's Test: p > 0.05 means variance is equal across groups (treatments)
leveneTest(residuals(mod_log) ~ df_analysis$treatment)
```

Variances are not equal, so we can either proceed with a huge amount of caution in interpretation or start to use conservative approaches. 

```{r}
anova(mod_log)
mod_log.out <- summary(mod_log)

total_variance <- sum(as_tibble(mod_log.out$varcor)$vcov)

ind_variance <- as_tibble(mod_log.out$varcor) %>%
  filter(grp == "ind_num") %>%
  pull(vcov)

# Interclass Correlation Coefficient (ICC)
ind_variance / total_variance
```

48.8% of all variation in egg diameter is simply due to differences between individuals...

```{r}
emm_log <- emmeans(mod_log, ~ treatment)
pairs(emm_log, adjust = "tukey")
```

All treatments are remain different from one another... and we have the issue of unequal variances.

## Going Further, the Conservative Route

Having all the treatments be statistically significant from one another is just down right weird. We know that the residuals of the model were skewed, even after the egg diameters were log transformed. Since the normality assumption is violated, the p-values are probably artificially deflated. We need to know if our treatment differences are real, so we can use a **generalized linear mixed model** (GLMM) with a link function.

```{r}
#| eval: true
#| include: true
#| warning: false
#| message: false
robust_mod <- glmer(egg_diameter_mm ~ treatment + (1|ind_num), family = Gamma(link = "log"), 
                    data = df_analysis)
library(car)
library(DHARMa)
Anova(robust_mod, type = "III")

residuals_sim <- simulateResiduals(fittedModel = robust_mod)
plot(residuals_sim)
```

::: {.callout-note}
The **Gamma** distribution called from `family =` tells the function to expect the skewed data. It adjusts the standard errors, so the pairwise comparisons might be more realistic. 

Read more about link functions on R-bloggers: [Generalized Linear Models understanding the link function](https://www.r-bloggers.com/2018/10/generalized-linear-models-understanding-the-link-function/)
:::

Normality problem is fixed with removing outliers, but there is an issue of variance. Linear mixed models are vulnerable to unequal variances. It is a good idea to move forward trying a more robust approach.

## Robust Model: `nmle` {#sec-robust-model}

```{r}
#| warning: false
#| message: false
# 1. Load the packages
library(nlme)
library(car)      # For Type III Anova
library(emmeans)  # For post-hoc pairwise comparisons
```

```{r}
# 2. Fit the model using Generalized Least Squares via lme
# random = ~ 1 | ind_num/treatment builds the nested random interaction term
# weights = varIdent(form = ~ 1 | treatment) models unique variances per group
nlme_model <- lme(fixed = egg_diameter_mm ~ treatment, 
                  random = ~ 1 | ind_num/treatment, 
                  weights = varIdent(form = ~ 1 | treatment), 
                  data = df_sub,
                  method = "REML")
```

```{r}
# 3. View the model summary (shows the estimated standard deviations per treatment)
summary(nlme_model)
```

```{r}
# 4. Generate the Type III ANOVA table 
# "marginal" in nlme is equivalent to Type III Sum of Squares
anova(nlme_model, type = "marginal")
# Anova(nlme_model, type = "III")
```

```{r}
# 5. Conduct post-hoc Tukey adjustments to see which treatments differ
treatment_means <- emmeans(nlme_model, ~ treatment)
pairwise_comps <- pairs(treatment_means, adjust = "tukey")
pairwise_comps
```

# Visualization

The best (statistically sound model) is the [robust model](#sec-robust-model) from `{nmle}`.

* Accomodates randomized block design with subsampling and unequal design
* Potential outliers were removed using a Cook's distance threshold
* Used `varIdent` function to address heteroscedasticity
* Individuals and treatment-by-individual interaction were modeled as nested random effects.
* Significant effect of treatment on egg diameter (Type III ANOVA)
* Post-hoc pairwise comparisons with Tukey adjustments
  - Iso70 had larger egg diameters than all other groups
  - No difference between EtOH80 and EtOH95
  - Formalin10 resulted in significantly smaller egg sizes than all other treatments.
